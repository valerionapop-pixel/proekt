# Мозг нашего сайта (Pawka / Лапка Shelter)

Единая точка правды для людей и ассистентов: что это за проект, где живёт контент и как устроены потоки данных.

---

## 1. Суть продукта

**Pawka Shelter** (`pawka-shelter` в `package.json`) — двуязычный (EN/RU) маркетингово-информационный сайт приюта: усыновление, помощь, истории, FAQ, контакты, блок доверия и политика конфиденциальности. Сейчас это в основном статический контент и демо-формы без бэкенда.

## 3. Где «мыслит» сайт: три опоры

1. **`src/i18n/translations.ts`** — весь пользовательский текст, тип `Dictionary`, локали `en` | `ru`, функция `resolveLocale(lang)` (fallback `en`), объект `dictionaries`. Любая новая фраза на сайте должна проходить через словарь и типы.
2. **`src/data/animals.ts`** — карточки животных: типы `AnimalKind`, `AnimalCard`, массив `animals` (имя, вид, породы EN/RU, возраст, прививки, `imageUrl`).
3. **Параметр URL `?lang=en` / `?lang=ru`** — выбор языка. Серверные страницы читают `searchParams`, резолвят локаль и передают `dict` в разметку; клиентские части (`Header`, `LanguageSwitcher`) дописывают `lang` к ссылкам.

Итого: **контент = словари + данные животных + query `lang`**. Отдельной CMS или БД в репозитории нет.

## 4. Карта маршрутов

Все страницы под `src/app/*/page.tsx`:

| Путь | Назначение |
|------|------------|
| `/` | Главная: hero (видео), быстрая форма (демо), плитки разделов |
| `/about` | О приюте |
| `/animals` | Каталог из `animals.ts` |
| `/adoption` | Шаги и условия усыновления |
| `/help` | Волонтёрство, передержка, транспорт, донаты |
| `/stories` | Истории усыновления (текст в словаре) |
| `/faq` | Вопросы-ответы |
| `/contact` | Контакты и форма (демо) |
| `/trust` | Прозрачность и уход |
| `/privacy` | Политика |

Корневой **`src/app/layout.tsx`**: глобальные `metadata` (title template, OG, alternates по `lang`). Замените `metadataBase` и описания, когда будет реальный домен.

---

## 5. Компоненты (`src/components`)

- **`Header`** — клиентский; навигация, бургер-меню, блокировка скролла; все ссылки с `?lang=…`.
- **`Footer`** — контакты и быстрые ссылки из словаря.
- **`LanguageSwitcher`** — клиентский; сохраняет pathname и остальные query, меняет только `lang`.
- **`ContactFormDemo`** — где подключён, используется как демо контактов.

Общая оболочка страниц: в типичном паттерне `Header` + контент + `Footer`, локаль и `dict` с сервера.

---

## 6. Статика и внешние ресурсы

- **`public/`** — логотип (`/images/pawka-logo-light.svg`), видео hero (`/videos/dogs.mp4`), прочие ассеты по путям из кода.
- **Картинки животных** — внешние URL (`cataas.com`, `placedog.net`); разрешены в `next.config.ts` через `images.remotePatterns`.

---

## 7. Как безопасно менять проект

| Задача | Куда смотреть |
|--------|----------------|
| Новый текст / перевод | `translations.ts` — оба языка + тип `Dictionary` |
| Новое животное | `animals.ts` — новый элемент массива, корректный `kind`, EN/RU породы |
| Новая страница | `src/app/<segment>/page.tsx` + пункты в `Header` nav и при необходимости в `Footer` / словаре `nav` |
| SEO / домен | `layout.tsx` → `metadataBase`, `alternates`, OG |
| Картинки с нового хоста | `next.config.ts` → `images.remotePatterns` |

## 9. Заметки на будущее

- Формы на главной и контактах сейчас **не отправляют** данные на сервер — перед продом нужен endpoint, валидация и политика персональных данных.
- `resolveLocale` принимает только `ru` как не-английский; любые другие коды падают в `en`.
- После смены бренда или юридического лица обновить блок **Trust** и **Privacy** в словарях.
